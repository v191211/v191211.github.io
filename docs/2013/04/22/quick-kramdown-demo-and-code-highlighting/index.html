    Jekyll 是一个用于个人、项目或组织站点，简单的、博客相关的静态站点生成器。

    可以理解为是一个 CMS （内容管理系统），但是没有那么复杂。Jekyll 获取内容，渲染 Markdown 和模板，然后输出完整的、可托管在 Apache、Nginx 或其它网站服务器上的静态网站。

    Jekyll 是 GitHub Pages 的引擎，可以托管 GitHub 仓库中的网站。
